🧪 Zone Fallback 테스트

Zone은 있지만 Record가 없을 때 Upstream으로 자동 Fallback되는지 테스트

1️⃣ Zone 생성

fallback.test. Zone 생성

2️⃣ Record 생성

api.fallback.test. A 레코드만 등록

3️⃣ DNS 쿼리 테스트

✅ Zone + Record 있음

dig @localhost -p 5300 api.fallback.test

기대: 10.0.0.100, flags에 "aa" 포함 (Authoritative)

🔄 Zone만 있음 (Fallback)

dig @localhost -p 5300 www.fallback.test

기대: Upstream 응답 또는 NXDOMAIN, flags에 "aa" 없음

🔍 서버 로그 확인

터미널에서 다음 로그 메시지를 확인하세요:

[DNS] Zone fallback.test. exists but no record for www.fallback.test. A, falling back to upstream
